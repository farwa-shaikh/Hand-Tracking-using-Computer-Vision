What a Hand Tracking Project Does (Main Functions)
1. Detects the Hand

The system identifies where the hand is in the camera frame.

Often uses deep-learning models like MediaPipe, OpenCV, or CNN-based detectors.

2. Tracks Hand Landmarks

It finds key points on the hand (fingertips, knuckles, wrist, etc.).

Typically around 21 landmark points are tracked.
